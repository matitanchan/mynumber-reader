# 券面事項入力補助AP読み出しPythonプログラム
マイナンバーカードの券面事項入力補助APから4情報(氏名・住所・生年月日・性別)を読み出すPythonプログラムです。

## ひつようなもの
・PaSoRi(nfcpyに対応したもの)
・パソコン
・マイナンバーカード

## ダウンロードが必要なもの
・Python3
・Zadig
・pip install nfcpy

## 使い方
実行すると暗証番号で認証するか照合番号Bで認証するか聞いてくるので選択してそのあとは画面の指示に従って入力すれば4情報を読み出せます。
